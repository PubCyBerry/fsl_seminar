mode 전환
esc, Ctrl [, Ctrl c: 노멀 모드로 전환
v: 비주얼 모드
: 커맨드 모드
입력 모드

insert mode
i, I: 커서 왼쪽, 줄 왼쪽
a, A: 커서 오른쪽, 줄 오른쪽
o, O: 아랫줄, 윗줄

replace with insert mode
r, R: 단일 글자 수정, 연속 수정
c, C: 액션 단위로 지우면서 입력 모드 진입, 커서 뒤로 전부 지우면서 입력 모드 진입
s, S: 단일 글자 지우면서 입력 모드 진입, 줄 지우면서 입력 모드 진입

copy, paste, delete
y, yy(Y): 액션 단위 복사, 줄 복사
p, P: 커서 뒤에 붙여넣기, 커서 앞에 붙여넣기
d, D: 액션 단위 삭제, 줄 삭제
x, X: 커서 뒤 / 앞 글자 지우기
Ctrl h: 백스페이스와 같음(vscode에선 안됨)

undo, redo
u: 마지막 변경사항을 취소함(Ctrl + z)
Ctrl r: u를 취소함(vscode에선 안됨)

browsing
h, j, k, l: 왼쪽, 아래, 위, 오른쪽
w, W: 단어 첫글자 단위 오른쪽 이동, 공백 단위 첫글자 오른쪽 이동
b, B: 단어 첫글자 단위 왼쪽 이동, 공백 단위 첫글자 왼쪽 이동
e, E: 단어 마지막 글자 단위 오른쪽 이동, 공백 단위 마지막 글자 오른쪽 이동
ge, gE: 단어 마지막 글자 단위 왼쪽 이동, 공백 단위 마지막 글자 왼쪽 이동
gg, G: 첫 줄로 이동, 마지막 줄로 이동
0, ^, $: 줄 맨앞/ 문장 맨 앞/ 줄 끝으로 이동
f, F: 글자 검색 단위로 오른쪽/왼쪽 이동(; 혹은 ,로 반복 이동)(커서가 글자 뒤에 있음)
t, T: 글자 검색 단위로 오른쪽/왼쪽 이동(; 혹은 ,로 반복 이동)(커서가 글자 앞에 있음)
/<string>: 단어 찾기(n, N으로 다음 단어 검색)
Ctrl e, Ctrl y: 아래/위로 한 줄 단위 스크롤
Ctrl d, Ctrl u: 아래/위로 반 페이지 단위 스크롤 
Ctrl f, Ctrl b: 아래/위로 한 페이지 단위 스크롤(vscode에선 안됨)
gt, gT: 다음 탭, 이전 탭

<lineNumber>gg: 해당 줄번호로 이동
[[, ]], [], ][: 문단 단위 이동
'': 최근 이동 곳으로 이동(수백 줄 단위 왔다갔다 할 때 유용)

save, quit in command mode
:w 저장
:wq 저장하고 나가기
:wq! 강제로 저장하고 나가기
:q 저장 안하고 나가기

visual mode
v, V: 글자 단위 비주얼 모드, 줄 단위 비주얼 모드
Ctrl v: 세로 단위 비주얼 모드

plugin
surround

Misc
.: 마지막 명령어 반복 실행
gc: 액션 단위로 주석
